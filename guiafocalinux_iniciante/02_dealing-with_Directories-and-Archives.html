Os comandos deverão ser executados com privilégios elevados apenas no caso de sua manipulação estiver restrita ao usuário mestre

cd - Change Directoory

Instrui o shell para atuar em outra pasta. Para acesso aos arquivos protegidos ou ocultos é necessário privilégio root.
  • $cd Documents/MyBkp/Photos

ls - List Segments

Esse comando é usado para listar todos os segmentos do seu diretório atual no terminal. Enxergar pastas protegidas ou arquivos ocultos somente como usuário root.
  • $ls -a #Quando ls não é executado em modo super-usuário, este será útil para listar arquivos ocultos.

pwd - Print Working Directory

O resultado desse comando imprimirá o caminho completo do atual direrório em operação no terminal.
  • $pwd
  • output>/home/User/Documents/MyBkp/Photos

cat - Concatenate

Esse comando mostra o conteúdo de um binário ou arquivo de texto.
  • $cat /etc/credentials #Exibe o conteúdo do arquivo.
  • $cat bomdia.txt boanoite.txt #Irá imprimir o conteúdo dos 2 arquivos.
  • $more some_huge_text_file #Exibe o conteúdo da mensagem com scroll habilitado.
  • $cat >novo.txt #Irá criar um novo arquivo do tipo texto.
  • $cat file0 file1 file2 > file3 #Combina 3 arquivos e gera um novo arquivo, irá sobrescrever arquivos com mesmo nome.
  • $cat -n code_file.rb #Exibe a quantidade de linhas impressas de um arquivo.
  • $cat -T houses.json #O parâmetro -T exibe o número de caracteres não impressos TAB.
  • $cat -t houses.sql #O parâmetro -t faz o mesmo que o -Tv, imprimindo a quantidade total de caracteres não impressos.

Extensões comuns em arquivos

Sabendo o tipo de arquivo facilitará a compreensão de suas características, importância, e opções de edição.
  • name.txt #A nomenclatura .txt nos revela se tratar de um arquivo em texto pleno.
  • name.sh #A nomenclatura .sh nos revela se tratar de um arquivo de script.
  • name.log #A nomenclatura .log nos revela se tratar de um registro de algum programa existente no sistema.
  • name.gz #A nomenclatura .gz nos revela se tratar de um arquivo compactado pelo utilitário gzip.
  • name.html #A nomenclatura .html nos revela se tratar de um arquivo de Hypertexto.

A extensão (na maioria dos casos) não é requerida pelo sistema operacional GNU/Linux, mas é conveniente o seu uso para determinarmos facilmente o tipo de arquivo e que programa precisaremos usar para abri-lo.
