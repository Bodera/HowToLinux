Os comandos deverão ser executado com privilégios elevados e suas opções são:

Shutdown

Agenda um tempo para o sistema ser desligado. Ele pode ser utilizado para travar, desligar ou reiniciar a máquina.
  • #shutdown #Desliga a máquina com delay de 30 segundos
  • #shutdown now #Desliga a máquina imediatamente
  • #shutdown 21:00 #Desliga a máquina no horário instruído
  • #shutdown -p #Poweroff
  • #shutdown -H #Halt
  • #shutdown -r #Reinicia a máquina
  • #shutdown -r10:05 #Reinicia a máquina às 10:05AM
  • #shutdown -c #Cancela um desligamento agendado
  • #shutdown -k #Envia alertas sem reiniciar, poweroff ou halt
  • #shutdown --no-wall #Não envia uma mensagem que bloqueia o monitor antes de halt, poweroff ou reinício
Se o argumento de tempo é usado, 5 minutos antes que o sistema desative, o arquivo /run/nologin é criado para garantir que o sistema não aceite mais logons.

Halt

Esse comando instrui o hardware a parar todas as funções de CPU sem desligar.
  • #shutdown --halt #Para a máquina
  • #hatl #Também para a máquina
  • #halt -p #Executa poweroff
  • #halt --reboot #Reinicia a máquina

Poweroff

A característica desse comando é que ele envia sinais ACPI com instruções para desligar o sistema.
  • #poweroff #Desliga a máquina
  • #poweroff --halt #Executa um halt
  • #poweroff --reboot #Reinicia a máquina

Reboot

Reinicia o sistema e aceita opções comuns
  • #reboot now
  • #reboot --halt
  • #reboot -p
